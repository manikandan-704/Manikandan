A fully functional, animated portfolio website featuring:

✅ **Modern Tech Stack:** React + Vite + Tailwind CSS + Framer Motion  
✅ **Responsive Design:** Perfect on mobile, tablet, and desktop  
✅ **Smooth Animations:** Scroll reveals, hover effects, typing animations  
✅ **Dark Theme:** Sleek, professional design  
✅ **GitHub Pages Ready:** Deploy in minutes  
✅ **Production Optimized:** Fast loading, code splitting  

## 🎯 Quick Start (Choose Your Path)

### Path 1: Automated Setup (Recommended)

**For macOS/Linux:**
```bash
cd manikandan-portfolio
chmod +x setup.sh
./setup.sh
npm run dev
```

**For Windows:**
```bash
cd manikandan-portfolio
setup.bat
```

### Path 2: Manual Setup

```bash
cd manikandan-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🆘 Quick Troubleshooting

**"npm: command not found"**  
→ Install Node.js from nodejs.org

**Build errors**  
→ Run `npm install` again

**GitHub Pages not working**  
→ Check DEPLOYMENT_GUIDE.md
